# 飲食控制AI系統

這是一個 AI 驅動的飲食管理系統，幫助使用者記錄飲食、分析營養，並提供智慧飲食建議，我的練習專案。

---

## 專案目標

本專案目的是建立一個完整的：

- 飲食紀錄系統
- 營養分析系統
- AI 飲食建議平台

---

## 功能（MVP）

- 使用者註冊 / 登入（JWT）
- 飲食紀錄（CRUD）
- 營養分析（熱量 / 蛋白質 / 碳水 / 脂肪）
- AI 飲食建議（OpenAI / Gemini）

---

## 技術棧

### Backend
- Python 3.12
- Django
- Django REST Framework
- PostgreSQL

### AI
- OpenAI API
- Google Gemini API

### DevOps
- Docker
- Docker Compose
